Gene Drives

Description

Today we started learning about gene drives and how they can be important to help cure diseases, and make sure an organism's offspring have modified genes, and have those be modified in all their offspring.

Claim, Support, Question

Gene drives allow a specific trait to always be passed down to offspring, thus passing it into the population. Scientists have been able to stop mosquito's ability to spread malaria. This is done via CRISPR, editing the cells of the offspring, leaving them with the specific trait. What I'm curious about though is how this can be used in another scenario / enviroment, and what else can this be used for?
